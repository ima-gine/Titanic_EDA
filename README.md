### Titanic_EDA

# Titanic Dataset - Exploratory Data Analysis (EDA)
This project focuses on performing an Exploratory Data Analysis (EDA) on the classic Titanic dataset. The goal is to uncover patterns and identify key factors that influenced passenger survival rates during the disaster.

## Objectives
 * Data Inspection: Examine the dataset structure, data types, and handle missing values.
 * Survival Analysis: Analyze how variables like Gender, Socio-economic Class (Pclass), and Age impacted survival.
 * Visualization: Utilize statistical plots to represent data distributions and correlations.
 * Insights: Derive actionable conclusions from the visual and statistical evidence.
   
## Key Insights
 * The Gender Gap: Survival rates were significantly higher for females compared to males, reflecting the "women and children first" protocol.
 * Class Influence: Passengers in First Class had a much higher probability of survival than those in Third Class, indicating a strong correlation between socio-economic status and safety access.
 * Age Distribution: Younger passengers (children) saw a higher survival rate in specific classes, though the majority of passengers were adults between 20 and 40.
